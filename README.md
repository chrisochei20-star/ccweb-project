# ccweb-project

The world's first AI-powered Web3 Academy and Business Engine.

Core pillars: Learn -> Find -> Build -> Earn.

## Development setup

### Prerequisites

- Node.js 18+ and npm

### Install dependencies

Run:

`npm install`

### Start the application

Run:

`npm run dev`

Then open `http://localhost:3000`.

### Production-style run

Run:

`npm start`
