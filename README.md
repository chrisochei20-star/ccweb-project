# CCWeb - Web3, Crypto & AI Education Platform

A modern education platform for Web3, cryptocurrency, and artificial intelligence, built with React, TypeScript, and Tailwind CSS.

## Features

- **Home** - Hero section, platform stats, featured courses, testimonials, and CTA
- **Courses** - Full course catalog with search, category, and level filtering
- **Blog** - Articles and insights with search and category filters
- **About** - Mission, values, team profiles, and milestones timeline
- **Contact** - Contact form, FAQ section, and contact information

## Tech Stack

- **React 19** with TypeScript
- **Vite** for fast development and building
- **Tailwind CSS v4** for styling
- **React Router** for client-side routing
- **Radix UI** primitives for accessible components
- **Lucide React** for icons

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
