# ccweb-project

Crypto, Web3, and AI education platform imported from the provided Lovable preview.

## Scripts

- `npm run dev` - start the Vite dev server
- `npm run build` - create a production build in `dist/`
- `npm run preview` - preview the production build locally

## Notes

- The app is served as a single-page application through Vite.
- Static assets imported from the preview live under `public/assets/`.
